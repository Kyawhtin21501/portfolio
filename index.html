自己紹介

中央情報専門学校

私は、AI と Web 技術を組み合わせたソリューション開発に注力しているフルスタック志向のエンジニアです。現在は 中央情報専門学校 にて、機械学習・統計モデリング・ソフトウェア開発を学んでいます。

HTML / CSS / JavaScript を用いた UI 実装や、Python（Flask, scikit-learn）による API 開発も手がけています。

AI を活用した業務支援アプリ、観光ガイドサービス、UI/UX を意識したミニアプリなど、実践的なプロジェクトに取り組んできました。

今後は AI の社会実装と Web 開発の融合をさらに深め、ユーザーにとって意味のある体験を提供していきたいと考えています。
